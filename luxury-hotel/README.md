## üöÄ How to Run Locally
1. **Clone the project:** `git clone https://github.com/Alphonso1068/luxury-hotel.git`
2. **Move to XAMPP:** Place the project folder inside `C:/xampp/htdocs/`.
3. **Setup Database:**
   - Open `phpMyAdmin`.
   - Create a new database named `luxury_hotel_db`.
   - Import the `database.sql` file provided in this repository.
4. **Configuration:**
   - Ensure your `db_config.php` matches your local database credentials.
5. **Launch:**
   - Open your browser and go to `http://localhost/luxury-hotel/`.
